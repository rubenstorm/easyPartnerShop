easyPartnerShop

easyPartnerShop is a shop system where you can set up a partnershop like Amazon

Description

With this you can include as many partner shops as you like in one installation on your webspace. It does not matter if Amazon or Spreadshirts. Just configure your shop and you a good to go.

Features

  • One Install -- Install once, use more shops
  • Own Design -- Make your own design
  • Single or Multi -- Design for all or each a other
  • Easy Setup -- Configure in simple files

You can install it once, have as many shops you like, all shops with the same design or every shop with his own
